== Description ==

Powerful and easy to use chat widget for any website - Chirpbox.

== Installation ==

To install:

1. Upload `chirpbox-chat` folder to the `/wp-content/plugins/` directory or install through the plugins search in the wordpress backend
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to `Settings` then `Chirpbox Chat`
4. Fill in your site url and click save.

A chirpbox chat widget will automatically appear on your site.

NB: You must have already registered an account at http://getchirpbox.com and navigated to http://admin.getchirpbox.com.

If you are unsure of your site url it will be supplied in your admin (http://admin.getchirpbox.com) under `Get Insert Code` in the wordpress section.
